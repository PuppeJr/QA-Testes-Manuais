## Critérios de Aceitação

### Login

- Usuário autenticado com sucesso.
- Redirecionamento para Dashboard.
- Sessão criada corretamente.

### Dashboard

- Dados exibidos corretamente.
- Navegação funcional.

### Perfil do Usuário

- Informações carregadas corretamente.
- Atualizações persistidas.
