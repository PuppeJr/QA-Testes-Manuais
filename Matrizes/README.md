# Matrizes

## Objetivo

Esta pasta reúne os artefatos de gestão utilizados durante o processo de Qualidade de Software (QA).

As matrizes permitem controlar a cobertura dos testes, rastrear evidências e priorizar funcionalidades críticas.

---

## Conteúdo

| Documento | Descrição |
| ---------- | --------- |
| Matriz-de-Rastreabilidade.md | Relaciona funcionalidades, casos de teste, execuções, evidências e bugs. |
| Matriz-de-Riscos.md | Classifica os módulos do sistema conforme impacto e prioridade para os testes. |

---

## Atualização

Os documentos desta pasta são atualizados continuamente durante a evolução do projeto.
