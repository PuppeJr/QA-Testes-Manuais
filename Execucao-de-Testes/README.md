# Execução de Testes

## Objetivo

Esta pasta contém os registros da execução dos testes realizados durante o processo de Garantia da Qualidade (QA).

Cada documento representa uma execução de teste, inspeção visual ou teste exploratório realizado no sistema.

---

## Estrutura

| Documento | Descrição |
| ---------- | --------- |
| ET-001-Inspecao-Visual-Login.md | Inspeção visual da tela de autenticação |
| Exploracao-Login.md | Teste exploratório da funcionalidade de login |

---

## Evidências

As evidências produzidas durante as execuções (capturas de tela, vídeos e registros) devem ser armazenadas na pasta **Evidencias**.

---

## Objetivo da Execução

Durante esta fase são registrados:

- ambiente utilizado;
- data da execução;
- casos de teste executados;
- resultados obtidos;
- defeitos encontrados;
- evidências coletadas;
- conclusão da execução.
