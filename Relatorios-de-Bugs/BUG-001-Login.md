# BUG-001 - Mensagem de erro não exibida após senha inválida

## Identificação

ID: BUG-001

Título: Sistema não exibe mensagem de erro para senha incorreta.

Módulo: Login

Severidade: Média

Prioridade: Alta

Status: Aberto

---

## Ambiente

* Navegador: Google Chrome
* Sistema Operacional: Windows 11
* Versão da Aplicação: 1.0

---

## Passos para Reproduzir

1. Acessar a tela de login.
2. Informar usuário válido.
3. Informar senha incorreta.
4. Clicar em Entrar.

---

## Resultado Atual

Sistema permanece na tela de login sem exibir mensagem de erro.

---

## Resultado Esperado

Sistema deve exibir mensagem informando que as credenciais são inválidas.

---

## Observações

Comportamento pode gerar dúvidas para o usuário e aumentar chamados de suporte.
