# Plano de Testes – Funcionalidade Login

## Objetivo

Validar o funcionamento da autenticação de usuários em uma aplicação web.

## Escopo

Garantir que usuários cadastrados possam acessar o sistema utilizando credenciais válidas e que mensagens de erro sejam apresentadas corretamente para situações inválidas.

## Funcionalidades Testadas

* Campo E-mail
* Campo Senha
* Botão Entrar
* Mensagens de erro
* Redirecionamento após autenticação

## Tipos de Teste

* Teste Funcional
* Teste de Interface
* Teste de Validação de Campos

## Critérios de Aprovação

* Login realizado com sucesso para usuários válidos.
* Mensagens de erro exibidas corretamente.
* Nenhum comportamento inesperado durante a autenticação.

## Ambiente de Teste

* Sistema Operacional: Windows 11
* Navegador: Google Chrome
